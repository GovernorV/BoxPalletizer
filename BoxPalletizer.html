Ключевые возможности

Оптимальная укладка

Программа автоматически подбирает оптимальную схему укладки ящиков в слое, стремясь получить максимальное количество ящиков при соблюдении ограничений по габаритам, свесам и устойчивости.

Поддерживаются два основных варианта укладки:

  • Стандартная укладка — ящики укладываются с минимальным зазором, схема простая и привычная для рабочих.
  • Спиральная укладка — ориентация ящиков чередуется, что повышает устойчивость паллета и равномерность распределения нагрузки.

Если оба варианта дают одинаковое количество ящиков в слое, пользователь может задать приоритет: использовать стандартную или спиральную схему по умолчанию.

Дополнительно программа учитывает кантование ящиков — возможность укладки:

  • в обычном положении (на основание),
  • на ребро,
  • на торец.

Это позволяет находить более эффективные схемы для нестандартной тары и максимально использовать площадь поддона.

Расчёт нагрузки и прочности ящиков

Программа выполняет расчёт нагрузки на нижние слои в соответствии с ГОСТ 9142-90 для тары из гофрокартона.

В процессе расчёта определяется фактическая нагрузка при штабелировании и сравнивается с допустимой прочностью ящика.

В результатах отображаются:

  • нагрузка на нижний ящик, кг и Н
  • допустимая нагрузка при штабелировании
  • коэффициент запаса прочности
  • сопротивление торцевому сжатию (ECT)
  • тип и марка гофрокартона

3D-визуализация

Понятная схема для работников склада. Четные и нечетные слои укладки отобрадаются разным цветом. Возможен поворот наждого слоя на 180 градусов относительно соседних слоев для более прочной укладки (связка слоев). На схему можно добавить текстовый блок с параметрами оптимальной укладки. Схему можно скопировать в буфер обмена и вставить в любой офисный документ (Word, Excel, PowerPoint)

Отчёт Word с шаблоном пользователя

Программа формирует подробный отчёт в формате Microsoft Word, который может быть настроен под шаблон пользователя.

Вы можете использовать собственный бланк предприятия: логотип, реквизиты, фирменные шрифты, структуру документа — отчёт автоматически заполняется рассчитанными данными.

Это удобно для передачи схем укладки в производство, логистику или заказчику без ручного оформления.
